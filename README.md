# flutter-music

基于flutter开发的一款高颜值的音乐APP（完整开源）
该项目是2019年初完成的，该项目有些provider方法可能过时了，可以自己改成最新的写法；所有的功能全部完善，只是播放哪里key（现在抓取不到）
